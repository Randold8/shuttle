Введите функцию поведения шаттла

Цель игры:

Главная задача игры - написать программу для успешной посадки шаттла на различные поверхности в ходе симуляций. Конечная цель - посадить настоящий шаттл с первыми колонизаторами на Марс.

Параметры игры:

Координаты шаттла по оси X (горизонтальное положение) можно получить с помощью MM.getData(1), по оси Y (вертикальное положение) – MM.getData(2).
Скорость шаттла по оси X (горизонтальная скорость) доступна через MM.getData(3), по оси Y (вертикальная скорость) - MM.getData(4).
Угол наклона шаттла обозначается переменной angle. Оставшееся количество топлива можно узнать с помощью MM.getData(5).
Угол наклона angle и мощность двигателя power - две переменные, которые можно изменять в программе. Угол ограничен значениями от -90 до 90 градусов и влияет на направление тяги двигателя.
Мощность двигателя power может принимать значения от 0 (бездействие) до 10 (максимальная тяга). Скорость шаттла зависит от гравитации планеты и направления тяги двигателя.
Поведение шаттла меняется после вызова return [power, angle] с указанием желаемых значений мощности и угла.
Гравитация на поверхности планеты составляет 3.71 единицы.

Параметры миссии:

Игра разделена на миссии для проверки универсальности программы. В каждой миссии шаттл стартует из разных положений, а посадку нужно осуществить на различные типы поверхности. Поверхность задана точками, соединенными линиями. Координаты точек можно получить через MM.getData(6), возвращающую список точек для текущей миссии. Каждая точка имеет координаты X и Y.
В каждой миссии есть плато - участок поверхности для посадки шаттла. Плато располагается между двумя соседними точками с одинаковой координатой Y.


Условия успешной посадки:

Приземление на плато.
Угол наклона: -10: < angle < 10.
Горизонтальная скорость: -5: < Sx < 5.
Вертикальная скорость: Sy < 5.
